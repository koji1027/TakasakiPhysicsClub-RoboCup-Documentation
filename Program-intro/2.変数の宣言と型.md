# 変数の宣言と型
C\C++では、変数を宣言して、宣言した変数に値を格納し、その値を取り出す(見る)ことができる。箱に名前を付けて、そこに値を入れておくイメージだ。この動作をするプログラムを以下に示した。

``` cpp
#include <stdio.h>

int main() {
    int a;

    a = 1;

    printf("%d\n", a);

    return 0;
}
```
